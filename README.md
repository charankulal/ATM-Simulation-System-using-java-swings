# ATM-Simulation-System-using--JAVA-Swings
ATM simulation System is a JAVA desktop applicated created solely using the java swings and mysql for database.

The Apache Netbeans is the prefered IDE for the project development.

JDK 17 is required.

For Navigating into class files first unzip the file and then ATM Simulation System->ATM Simulation System->src->atm->simulation->system->"Bingo"
